## 模板

广度优先遍历也是层序遍历，使用队列，将各节点分层入队，分层遍历

步骤：

- 循环遍历各层节点
- 遍历节点时将其子节点依次入队

伪代码如下：

```java
// 首先将根节点入队
queue.offer(root);
while(!queue.isEmpty()) {
    // 获取本层节点数
    int size = queue.size(); 
    // 遍历本层节点
    for(node : queue) { 
        // do sth
        // 将下一层节点（即子节点）入队，
        queue.offer(node.left); 
        queue.offer(node.right); 
    }
}
```
